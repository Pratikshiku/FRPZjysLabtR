### 三、功能描述
系统分为（身份）：普通用户、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 汽车配件
5. 汽车资讯
6. 后台管理
7. 个人中心
8. 我的收藏

**员工功能：**
1. 登录
2. 系统首页（销售业绩、销售品牌统计、日销售额统计）
3. 修改密码
4. 个人信息
5. 热销汽车管理
6. 热销汽车
7. 订单信息管理
8. 汽车配件管理
9. 配件订单管理
10. 售后信息管理
11. 潜在客户管理
12. 系统管理（汽车资讯）



**管理员功能：**
1. 登录
2. 系统首页（销售业绩、销售品牌统计、日销售额统计）
3. 个人中心（修改密码、个人信息）
4. 用户管理
5. 员工管理
6. 部门信息管理
7. 岗位信息管理
8. 汽车品牌管理
9. 热销汽车管理
10. 订单信息管理
11. 配件分类管理
12. 汽车配件管理
13. 配件订单管理
14. 售后信息管理
15. 潜在客户管理
16. 系统管理（轮播图管理、汽车资讯、系统简介）
